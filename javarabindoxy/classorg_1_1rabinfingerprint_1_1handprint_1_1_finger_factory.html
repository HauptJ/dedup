Fingerprint the file into chunks called "Fingers". The chunk boundaries are determined using a windowed fingerprinter RabinFingerprintLongWindowed.

The chunk detector is position independent. Therefore, even if a file is rearranged or partially corrupted, the untouched chunks can be efficiently discovered.
